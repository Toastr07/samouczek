Samouczek HTML5 i CSS od podstaw

Tworzenie stron internetowych opiera się na dwóch technologiach. Język HTML odpowiada za treść i strukturę witryny, natomiast arkusze stylów CSS nadają jej wygląd, kolory oraz układy. Ten poradnik w prosty sposób wyjaśnia działanie obu tych narzędzi na praktycznych przykładach kodu.

O czym jest ten samouczek?

W części poświęconej HTML5 nauczysz się budować strukturę strony, formatować teksty, tworzyć listy, tabele, linki oraz formularze. Poznasz także znaczniki semantyczne, które pomagają w pozycjonowaniu witryny w wyszukiwarkach.

W części dotyczącej CSS dowiesz się, jak stylizować elementy za pomocą klas i identyfikatorów. Omówimy model pudełkowy, ustawienia tła, projektowanie menu, tworzenie stron responsywnych na telefony oraz płynne animacje.
