Lekcja 12: Obramowania i tabele

Warstwa obramowania (Border) jest integralną częścią każdego modelu pudełkowego. Za pomocą stylów CSS możemy nie tylko kontrolować wygląd, grubość i kolor krawędzi, ale również wykorzystać te właściwości do estetycznego sformatowania tabel poznanych w części HTML5.

Obramowanie elementów (Właściwości border)

Obramowanie pozwala na rysowanie linii wokół elementów oraz modyfikowanie ich kształtu. Składa się z trzech podstawowych parametrów, które możemy zapisać w jednej linii za pomocą właściwości skróconej border: grubość styl kolor;:

Grubość ramki definiujemy najczęściej w pikselach (np. 2px).

Styl ramki określa jej wygląd. Najważniejsze style to: solid (linia ciągła), dashed (linia kreskowana), dotted (linia kropkowana) oraz none (brak obramowania).

Kolor ramki podajemy słownie lub za pomocą kodu szesnastkowego (np. black lub #333).

Możemy stylizować każdą krawędź osobno za pomocą właściwości kierunkowych: border-top (góra), border-bottom (dół), border-left (lewa) oraz border-right (prawa).

Właściwość border-radius służy do zaokrąglania rogów obramowania oraz całego pudełka. Wartość podana w pikselach (np. 10px) zaokrągla rogi delikatnie, natomiast wartość 50% pozwala na stworzenie idealnego koła z kwadratowego elementu.

Stylowanie tabel w CSS

Tabele domyślnie wyświetlają się w przeglądarce bez ramek lub z nieestetycznymi podwójnymi liniami. Za pomocą właściwości stylów możemy w pełni kontrolować ich wygląd:

Właściwość border-collapse: collapse; zadeklarowana na znaczniku <table> usuwa puste przestrzenie między komórkami i scala podwójne linie ramek w jedną pojedynczą linię.

Właściwości padding oraz border przypisane do komórek <th> oraz <td> pozwalają na ustawienie grubości ramki i odsunięcie tekstu od jej krawędzi, co diametralnie poprawia czytelność danych.

Selektor tr:nth-child(even) (lub odd) pozwala na automatyczne nadanie innego koloru tła co drugiemu wierszowi w tabeli (tzw. efekt zebry), co ułatwia śledzenie wzrokiem szerokich danych.

<!DOCTYPE html> <html lang="pl"> <head> <meta charset="UTF-8"> <title>Stylowanie obramowań i tabel</title> <style> .blok-pokazowy { border: 3px dashed #333; border-radius: 15px; padding: 20px; background-color: #fafafa; } table { width: 100%; border-collapse: collapse; background-color: white; margin-top: 10px; } th, td { border: 1px solid #333; padding: 8px; text-align: left; } th { background-color: #2c3e50; color: white; } tr:nth-child(even) { background-color: #f2f2f2; } </style> </head> <body> <div class="blok-pokazowy"> <table> <thead> <tr> <th>Przedmiot</th> <th>Ocena</th> </tr> </thead> <tbody> <tr> <td>Podstawy HTML5</td> <td>Bardzo dobry</td> </tr> <tr> <td>Arkusze CSS3</td> <td>Celujący</td> </tr> </tbody> </table> </div> </body> </html>
