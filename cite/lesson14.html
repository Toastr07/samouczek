Lekcja 14: Menu poziome i pionowe

Domyślnie elementy listy <li> są elementami blokowymi i układają się pionowo jeden pod drugim. W zależności od projektu witryny, nawigację możemy przygotować w formie klasycznego, bocznego menu pionowego lub nowoczesnego, poziomego paska za pomocą elastycznego modelu układu o nazwie Flexbox.

Nawigacja pozioma za pomocą Flexboxa

Właściwość display: flex; zadeklarowana na liście nadrzędnej <ul> zmienia domyślny kierunek układania elementów. Sprawia ona, że pozycje menu automatycznie ustawiają się w jednej poziomej linii obok siebie.

Właściwość list-style-type: none; usuwa standardowe kropki wypunktowania z listy, a reset marginesów pozwala na równe rozłożenie paska.

Nawigacja pionowa (Blokowa)

W menu pionowym elementy celowo zostawiamy jeden pod drugim. Często ogranicza się wtedy szerokość całego menu za pomocą właściwości width na kontenerze nadrzędnym.

W obu wariantach kluczowe jest przekształcenie linków w elementy blokowe za pomocą display: block; oraz dodanie odstępów padding. Gwarantuje to, że klikalny staje się cały obszar przycisku menu, a nie tylko sam tekst.

<!DOCTYPE html> <html lang="pl"> <head> <meta charset="UTF-8"> <title>Menu Poziome i Pionowe</title> <style> .menu-poziome, .menu-pionowe { list-style-type: none; padding: 0; margin: 0; } .menu-poziome li a, .menu-pionowe li a { display: block; color: white; padding: 10px 15px; text-decoration: none; } .menu-poziome li a:hover, .menu-pionowe li a:hover { background-color: #555; } .menu-poziome { background-color: #333; display: flex; margin-bottom: 30px; } .menu-pionowe { background-color: #2c3e50; width: 200px; } .menu-pionowe li { border-bottom: 1px solid #34495e; } .menu-pionowe li:last-child { border-bottom: none; } </style> </head> <body> <h3>Przykład menu poziomego:</h3> <ul class="menu-poziome"> <li><a href="#">Start</a></li> <li><a href="#">Galeria</a></li> <li><a href="#">Kontakt</a></li> </ul> <h3>Przykład menu pionowego:</h3> <ul class="menu-pionowe"> <li><a href="#">Profil</a></li> <li><a href="#">Ustawienia</a></li> <li><a href="#">Wyloguj</a></li> </ul> </body> </html>

Przykład działania

Poniżej możesz zaobserwować oba warianty nawigacji wyrenderowane obok siebie. Menu poziome zajmuje pełną szerokość i świetnie sprawdza się na górze strony, natomiast menu pionowe ma ograniczoną szerokość do 200 pikseli i posiada delikatne linie oddzielające poszczególne opcje:

Podgląd menu poziomego:

  • Start
  • Galeria
  • Kontakt

Podgląd menu pionowego:

  • Profil
  • Ustawienia
  • Wyloguj
