Lekcja 10: Wprowadzenie do CSS

CSS (Cascading Style Sheets) odpowiada za wygląd i oprawę wizualną strony internetowej. Podczas gdy HTML tworzy szkielet i treść, CSS nadaje jej kolory, wielkość czcionek i odpowiednie układy. Reguła CSS składa się z selektora oraz bloku stylów zamkniętego w klamrach.

Budowa reguły CSS

Styl definiujemy za pomocą zapisu: selektor { właściwość: wartość; }.

Selektor znacznika (np. p) odnosi się do wszystkich akapitów na stronie.

Selektor klasy oznaczamy w pliku CSS kropką (np. .zielony) i odnosi się on do elementów z atrybutem class="zielony".

Właściwość color definiuje kolor tekstu, a właściwość font-size odpowiada za wielkość czcionki.

<!DOCTYPE html> <html lang="pl"> <head> <meta charset="UTF-8"> <title>Wprowadzenie do CSS</title> <style> p { font-size: 16px; } .zielony { color: green; font-weight: bold; } </style> </head> <body> <p>Jestem zwykłym tekstem.</p> <p class="zielony">Jestem tekstem z klasą zielony.</p> </body> </html>

Przykład działania

Jestem zwykłym tekstem.

Jestem tekstem z klasą zielony.
