Lekcja 4: Listy

Listy służą do porządkowania danych i grupowania powiązanych ze sobą elementów. W HTML5 wyróżniamy trzy typy list: nieuporządkowane, uporządkowane oraz listy opisowe (definicji). Każda pozycja na liście musi być zamknięta w osobnym znaczniku.

Typy list i ich elementy

Znacznik <ul> definiuje listę nieuporządkowaną. Przeglądarka automatycznie dodaje wcięcie w tekście oraz oznacza elementy standardowymi kropkami.

Znacznik <ol> definiuje listę uporządkowaną. Przeglądarka sama generuje numery dla kolejnych pozycji.

Atrybut type w liście uporządkowanej zmienia styl numeracji (1, a, A, i, I), a atrybut start definiuje liczbę początkową.

Znacznik <li> to pojedynczy element listy. Może występować wyłącznie wewnątrz znacznika <ul> lub <ol>.

Znacznik <dl> inicjuje listę opisową, służącą do tworzenia słowników lub spisów pojęć.

Znacznik <dt> oznacza pojęcie lub termin w liście opisowej.

Znacznik <dd> zawiera właściwy opis lub definicję pojęcia zadeklarowanego w wyższym elemencie.

<!DOCTYPE html> <html lang="pl"> <head> <meta charset="UTF-8"> <title>Przykłady list</title> </head> <body> <h3>Lista zakupów (wypunktowana z zagnieżdżeniem)</h3> <ul> <li>Pieczywo</li> <li>Owoce <ul> <li>Jabłka</li> <li>Banany</li> </ul> </li> </ul> <h3>Podium wyścigu (rzymskie cyfry)</h3> <ol type="I" start="1"> <li>Miejsce pierwsze</li> <li>Miejsce drugie</li> </ol> <h3>Słowniczek pojęć (opisowa)</h3> <dl> <dt>HTML</dt> <dd>Język znaczników do tworzenia struktury stron internetowych.</dd> <dt>CSS</dt> <dd>Arkusz stylów odpowiedzialny za warstwę wizualną.</dd> </dl> </body> </html>

Przykład działania

Lista zakupów (wypunktowana z zagnieżdżeniem)

  • Pieczywo
  • Owoce
    • Jabłka
    • Banany

Podium wyścigu (rzymskie cyfry)

  1. Miejsce pierwsze
  2. Miejsce drugie

Słowniczek pojęć (opisowa)

HTML
Język znaczników do tworzenia struktury stron internetowych.
CSS
Arkusz stylów odpowiedzialny za warstwę wizualną.
