Lekcja 16: Animacje

Najprostszą i najbardziej wydajną formą animacji w CSS jest właściwość płynnego przejścia wartości. Pozwala ona na łagodną zmianę wyglądu elementu w określonym przedziale czasowym, zamiast zmiany natychmiastowej.

Właściwość transition i pseudoklasa hover

Pseudoklasa :hover aktywuje się automatycznie w momencie, gdy użytkownik umieści kursor myszy nad danym elementem na stronie.

Właściwość transition informuje przeglądarkę, że zmiana konkretnego stylu ma trwać określony czas (np. 0.5s czyli pół sekundy) i odbywać się z naturalną płynnością ease.

<!DOCTYPE html> <html lang="pl"> <head> <meta charset="UTF-8"> <title>Animacje i przejścia</title> <style> .przycisk { background-color: black; color: white; padding: 10px 20px; border: none; cursor: pointer; transition: background-color 0.5s ease; } .przycisk:hover { background-color: red; } </style> </head> <body> <button class="przycisk">Najedź na mnie</button> </body> </html>
