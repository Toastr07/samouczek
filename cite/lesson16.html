Lekcja 16: Płynne przejścia i animacje

CSS oferuje zaawansowane narzędzia do dynamicznej zmiany wyglądu elementów na stronie internetowej. Ruch możesz realizować na dwa sposoby: za pomocą płynnych przejść stanów (właściwość transition) lub poprzez tworzenie złożonych, zapętlonych animacji klatkowych (reguła @keyframes).

Płynne przejścia (transition)

Właściwość transition pozwala na łagodną zmianę stylów w określonym czasie (np. po najechaniu myszką), zamiast zmiany natychmiastowej.

Pseudoklasa :hover aktywuje się automatycznie w momencie, gdy użytkownik umieści kursor myszy nad danym elementem na ekranie.

Zapis skrócony transition: background-color 0.5s ease; informuje przeglądarkę, że zmiana koloru tła ma trwać pół sekundy i odbywać się w sposób naturalny.

Animacje klatkowe (@keyframes)

Reguła @keyframes pozwala definiować własne animacje i precyzyzyjnie kontrolować ich przebieg w procentach (od 0% do 100%).

Właściwość animation-name wskazuje nazwę animacji, którą wcześniej utworzyłeś za pomocą reguły @keyframes.

Właściwość animation-duration określa łączny czas trwania jednego pełnego cyklu animacji podawany w sekundach.

Właściwość animation-iteration-count definiuje liczbę powtórzeń. Wartość infinite sprawia, że animacja działa bez końca.

<!DOCTYPE html> <html lang="pl"> <head> <meta charset="UTF-8"> <title>Animacje i przejścia</title> <style> .przycisk { background-color: #2c3e50; color: #ffffff; padding: 10px 20px; border: none; cursor: pointer; transition: background-color 0.5s ease; } .przycisk:hover { background-color: #e74c3c; } .pulsujacy-blok { width: 100px; height: 100px; background-color: #3498db; animation-name: pulsowanie; animation-duration: 2s; animation-iteration-count: infinite; } @keyframes pulsowanie { 0% { transform: scale(1); background-color: #3498db; } 50% { transform: scale(1.1); background-color: #9b59b6; } 100% { transform: scale(1); background-color: #3498db; } } </style> </head> <body> <button class="przycisk">Najedź na mnie</button> <div class="pulsujacy-blok"></div> </body> </html>
