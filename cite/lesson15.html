Lekcja 15: Strony responsywne

Responsywność (Responsive Web Design) to technika tworzenia stron, które automatycznie dopasowują swój układ, wielkości elementów oraz czcionek do szerokości ekranu urządzenia (np. komputera lub telefonu). Wykorzystuje się do tego reguły warunkowe.

Zapytania mediów (Media Queries)

Podstawą stron responsywnych jest reguła @media. Pozwala ona na zaaplikowanie konkretnych stylów CSS tylko wtedy, gdy ekran spełnia określony warunek szerokości.

Zapis (max-width: 600px) informuje przeglądarkę, że style zapisane wewnątrz klamer mają uruchomić się wyłącznie na ekranach o szerokości mniejszej lub równej 600 pikseli.

<!DOCTYPE html> <html lang="pl"> <head> <meta charset="UTF-8"> <title>Strona Responsywna</title> <style> .kontener { width: 80%; background-color: blue; color: white; padding: 20px; } @media (max-width: 600px) { .kontener { width: 100%; background-color: red; } } </style> </head> <body> <div class="kontener">Zmień szerokość okna</div> </body> </html>
